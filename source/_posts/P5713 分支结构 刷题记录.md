这题是分支结构题单的一题，[题面地址](https://www.luogu.com.cn/problem/P5713)

本来是想了一下要不要算完两个方式再比较大小，但是发现可以直接手算解决问题，那就列了个方程。
$$
\begin{cases}
y_1=5x,\\
y_2=3x+11
\end{cases}
$$
联立方程组，解得x=5.5。故当x小于5.5时，Local较快；x大于5.5时，Luogu较快。

代码如下：
```cpp
#include <stdio.h>

int main()
{
 int n;
 scanf("%d", &n);

 if (n < 5.5) printf("Local");
 if (n > 5.5) printf("Luogu");

 return 0;
}
```
