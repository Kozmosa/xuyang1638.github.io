题目地址：https://www.luogu.com.cn/problem/P5705
这是洛谷官方顺序结构题单中的一题，可以使用字符串翻转知识求解。
代码如下（C++ O2）：
```cpp
#include <stdio.h>

int main() {
    char input[5];
    char output[5];
    scanf("%s", input);
    for(int i = 0; i < 5; i++) {
        output[4-i] = input[i];
    }
    printf("%s", output);

    return 0;
}
```
